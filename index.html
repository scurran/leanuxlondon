Welcome to the Deciding in the Browser page on GitHub.

There are bunch of files here you can use to get started quickly but feel free to add your own.

Download the files

Grab a zip of all the files from the link on this page or clone this directory from GitHub.

Inside you'll find a copy of the brief plus a folder called Tools with a bunch of javascript, css and html frameworks.

Create a new folder

Create a new folder for your team inside the root of leanuxlondon for example leanuxlondon/team-nigella.

Create your prototype in here and copy or add any assets you need in here.
